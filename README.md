# TripFit - 여행지 날씨 기반 옷차림 추천 웹사이트

## 📝 프로젝트 소개
TripFit은 여행지의 현재 날씨 정보를 바탕으로 적절한 옷차림을 추천해주는 웹 서비스입니다.

## 🌟 주요 기능
- 도시명 자동완성 검색
- 현재 날씨 정보 표시 (온도, 습도, 날씨 상태)
- 날씨 기반 옷차림 추천
- 현지 환율 정보 제공
  - 현지 환율 차트 보여주기 (구현 대기)
- 현지 시각 및 한국과의 시차 표시

## 🛠 사용 기술
- HTML5
- CSS3
- Vanilla JavaScript
- OpenWeatherMap API
- ExchangeRate API
- RestCountries API
